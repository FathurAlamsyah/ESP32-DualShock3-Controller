# ESP32 DualShock 3 Controller

This project allows an ESP32 to connect with a PS3 DualShock 3 controller using Arduino.

The repository contains source code and libraries that can be used for various wireless control projects such as:
- Wireless Robots
- Transporter Robots
- RC Projects
- Smart Vehicle Projects
- Custom ESP32 Automation Systems

## Features

- ESP32 and DualShock 3 wireless connection
- Arduino-based source code
- Easy integration into custom projects
- Real-time controller input reading
- Suitable for robotics and IoT applications

## Hardware Requirements

- ESP32 Development Board
- PS3 DualShock 3 Controller
- USB Cable (for initial pairing if needed)
- Motor Driver (optional for robot projects)

## Software Requirements

- Arduino IDE
- ESP32 Board Package
- Required PS3 Controller Libraries
